NAME

i.eodag.py - Eodag interface to install imagery datasets from various providers.

KEYWORDS

eodag, sentinel, landsat, modis, datasets, download

SYNOPSIS

i.eodag.py
i.eodag.py --help
i.eodag.py [-l] dataset=string [output=name] [provider=string] [start=string] [end=string] [relation=string] [clouds=integer] [timeout=integer] [--help] [--verbose] [--quiet] [--ui]

Flags:

-l
List the search result without downloading
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

dataset=string [required]
Imagery dataset to search for
Default: S1_SAR_GRD
output=name
Name for output directory where to store downloaded data OR search results
provider=string
Provider to use for searching/downloading, if none is provided the searching will be done according to the config file priority
start=string
Start date ('YYYY-MM-DD'), by default it is 60 days ago
end=string
End date ('YYYY-MM-DD')
relation=string
Relation with area of interest
Options: intersects, contains, within
Default: intersects
clouds=integer
Maximum cloud cover percentage for Landsat scene
timeout=integer
Download timeout in seconds
Default: 300
