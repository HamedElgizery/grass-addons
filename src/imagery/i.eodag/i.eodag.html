DESCRIPTION

WARNING: I.EODAG IS STILL UNDER DEVELOPMENT. THIS IS AN EXPERIMENTAL VERSION.

i.eodag allows to search and download imagery datasets, e.g. Sentinel, Landsat, and MODIS, from a number of different providers. The module utilizes the EODAG API, as a single interface to search for datasets on the providers that EODAG supports.

To be able to download data through i.eodag, you will need to register for the providers you are interested in. See providers registration, and configure EODAG for more details about configuring the providers' credentials.

Currently, only products which footprint intersects the current computational region extent (area of interest, AOI) are retrieved.

To only list available scenes, l flag must be set. If no start or end dates are provided, the module will search scenes from the past 60 days.

To download all scenes found within the time frame provided, the user must remove the l flag and provide an output directory. Otherwise, files will be downloaded into /tmp directory. To download only selected scenes, one or more IDs must be provided through the id option file and an output directory.

EXAMPLES

Search and list the available Sentinel 2 scenes in the Copernicus Data Space Ecosystem: 
i.eodag -l start=2022-05-25 end=2022-06-01 \
    dataset=S2_MSI_L2A provider=cop_dataspace



Download all available scenes:


i.eodag  start=2022-05-25 end=2022-06-01 \
    dataset=S2_MSI_L2A provider=cop_dataspace



Download selected scenes by ID:


i.eodag provider=cop_dataspace \
    id=S2B_MSIL2A_20240526T080609_N0510_R078_T37SDD_20240526T094753,S2B_MSIL2A_20240529T081609_N0510_R121_T37SED_20240529T124818 \
    -e config=custom/eodag_config.yaml
